# Herbalé

Tienda online de hierbas orgánicas con propósito. Sitio estático desplegado en Vercel.

## Desarrollo

Es un sitio estático de un solo archivo (`index.html`). Para previsualizarlo localmente:

```bash
python3 -m http.server 3000
# o
npx serve .
```

Luego abre http://localhost:3000.

## Despliegue

Cada push a `main` se despliega automáticamente en Vercel.
